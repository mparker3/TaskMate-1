# TaskMate - The to-do list. But better.

### Installation guide:

1. Install virtual environment in Python
```bash
$ pip install virtualenv
```

2. Set up virtualenv
```bash
$ virtualenv venv
```

3. Install dependencies
```bash
$ pip install -r requirements.txt
```

4. Run virtual environment
```bash
$ source venv/bin/activate
```

5. Run the app
```bash
$ chmod 777 run.py
$ ./run.py
```
